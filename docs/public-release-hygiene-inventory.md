## Verification Helper

`pnpm public-release:hygiene` now scans tracked file paths for clear local artifact classes, secret-looking filenames, tracked real `wrangler.jsonc`, and tracked real `public/ads.txt`. It prints paths only, never file contents or secret values, and exits nonzero only for clear forbidden tracked artifacts.

`pnpm public-release:history-audit` performs a non-destructive path and commit-subject history scan. It prints summaries only and does not rewrite history.
